 Dither Tool

Simple and customizable image dithering tool focused on clarity and ease of use.



 Overview

Dither Tool is a lightweight browser-based application for converting images using different dithering techniques and custom color palettes.

It is designed to be straightforward, flexible, and easy to modify.


 Features

 Multiple dithering algorithms:

   Floyd–Steinberg
   Atkinson
   Jarvis–Judice–Ninke
   Stucki
Ordered dithering:

  Bayer 4×4
  Bayer 8×8
Custom color palette support
Built-in retro palettes (Game Boy, NES, C64, etc.)
Automatic palette generation (K-Means)
Adjustable resolution for pixel-style output
Strength control for the dithering effect
Export result as PNG
Import/export palettes as JSON

Usage

1. Open the tool in your browser
2. Upload an image
3. Choose or create a color palette
4. Select a dithering algorithm and settings
5. Click "Process Image"
6. Download the result

 Running the project

No installation required.

Just open the `index.html` file in any modern web browser.


The project is fully client-side and easy to modify.

You can:

Add new dithering algorithms
Modify existing paletes
Adjust UI styles
Extend functionality as needed

Notes

 Processing is done locally in the browser
 Performance may vary depending on image size
